## 🎯 Sobre o Projeto

Este projeto é um guia completo para trabalhar com **Pandas**, a biblioteca Python mais utilizada para análise, limpeza e manipulação de dados. Os notebooks contêm exemplos práticos desde conceitos básicos até técnicas avançadas.

### Para quem é?

- 👨‍💻 Desenvolvedores iniciantes em análise de dados
- 📚 Estudantes de Data Science
- 🔬 Cientistas de dados em formação
- 📈 Profissionais que desejam aprender Pandas

## 🚀 Instalação

### Requisitos
- Python 3.7+
- pip ou conda

### Via pip
```bash
pip install pandas
```

### Via conda
```bash
conda install -c conda-forge pandas
```

### Com dependências adicionais recomendadas
```bash
pip install pandas numpy matplotlib seaborn jupyter
```

## 💻 Como Usar

1. **Clone o repositório:**
```bash
git clone https://github.com/OseasMonteiro/Trabalhando-com-Pandas.git
cd Trabalhando-com-Pandas
```

2. **Instale as dependências:**
```bash
pip install -r requirements.txt
```

3. **Inicie o Jupyter Notebook:**
```bash
jupyter notebook
```

4. **Abra os notebooks e explore os exemplos:**
Navegue pelos arquivos `.ipynb` e execute as células para aprender na prática.

## 📚 Conteúdo

Os notebooks cobrem os seguintes tópicos:

### Básico
- Introdução ao Pandas
- Criação e estrutura de DataFrames
- Leitura e escrita de arquivos (CSV, Excel, JSON)
- Inspeção de dados

### Intermediário
- Limpeza e tratamento de dados
- Transformação e reshape de dados
- Indexação e seleção de dados
- Operações com grupos (groupby)
- Merge e join de DataFrames

### Avançado
- Análise exploratória de dados (EDA)
- Manipulação de séries temporais
- Otimização de performance
- Uso de métodos estatísticos
